# Basic Example
## Quick Start
```
git clone https://github.com/algolab-inc/tensorflow-deep-q-network.git
cd tensorflow-deep-q-network/basic
python test.py -m models/trained/model.ckpt
```

## Usage
### Train
```
python train.py
```

### Test
```
python test.py
```

### Test with trained model
```
python test.py -m models/trained/model.ckpt
```

## Requirements
* TenforFlow
* Matplotlib
